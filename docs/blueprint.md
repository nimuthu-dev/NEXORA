# **App Name**: NEXORA

## Core Features:

- Nexora AI Assistant: A conversational learning companion that acts as a tool to summarize lecture notes, explain complex IIT syllabus concepts, and generate personalized revision checklists.
- The Resource Vault: A structured cloud repository utilizing Firestore to organize lecture notes, lab sheets, and past papers by year and semester with advanced search functionality.
- Mock Arena Simulator: An interactive exam environment featuring timed PDF papers and MCQ modules with automated scoring and difficulty levels saved to a database.
- Gamified Progress Engine: An addictive tracking system for daily study streaks, XP accumulation, and level-based achievement badges for consistent learning activities.
- GPA Analytics & Tracker: Visual performance monitoring that calculates semester-wise and overall GPA using interactive charts and grade-tracking tables.
- Universal Search & Command: A high-performance global search bar to instantly find resources, modules, mock exams, or assignments across the entire platform.
- AI-Generated Quiz Creator: An automated system that functions as a tool to ingest module material and generate relevant mock quiz questions to test student knowledge instantly.

## Style Guidelines:

- The design employs a dark-first approach with a Deep Space background (#0B1020), a Soft Cobalt primary color (#7C9CF5), and an Electric Cyan accent (#67E8F9) for a futuristic look.
- A secondary palette using Dreamy Lavender (#A78BFA) and muted slate for textual depth (#94A3B8) creates high contrast and readability on dark surfaces.
- Recommendation: 'Inter', a grotesque sans-serif, used throughout for both headlines and body text to provide a modern, machined, and neutral aesthetic. Note: currently only Google Fonts are supported.
- Thin, minimalist line icons inspired by high-end developer tools like Linear and Arc Browser, emphasizing clarity and tech-forward design.
- Floating UI layout featuring glassmorphism surfaces with high transparency, generous 24px corner rounding, and multi-layered depth using premium shadows.
- Premium motion design utilizing smooth page transitions, floating card elevations on hover, and animated XP progress bars to mimic the feel of an OS.
